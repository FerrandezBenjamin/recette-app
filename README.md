## Installation

# 1 . Installation des dépendances
```bash
composer install
```

# 2. Création du fichier d'env
```bash
cp .env.example .env
```

# 3. Génération clef app
```bash
php artisan key:generate
```

# 4. Exécuter les migrations
```bash
php artisan migrate
```

# 5. Feed BDD
```bash
php artisan db:seed
```
